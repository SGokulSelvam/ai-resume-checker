# AI Resume ATS Checker

An AI-powered tool that analyzes resumes against job descriptions and calculates an ATS score.

## Features

* Resume PDF upload
* Job description analysis
* ATS score calculation
* Skill detection
* Missing keyword analysis
* Resume improvement suggestions
